# CO-OP Curriculum Overview
## Modern Full-Stack Development Practicum
### SOTA 2026 Edition

---

## Program Structure

**Duration:** 4 Weeks (160 hours total)
**Format:** Hybrid (40% classroom / 60% hands-on lab)
**Prerequisites:** Basic computer literacy, Git fundamentals
**Outcome:** Job-ready full-stack developer with SOTA 2026 tool mastery

---

## Week-by-Week Roadmap

| Week | Focus | Key Deliverables | Tools |
|------|-------|------------------|-------|
| **1** | Foundation | IDE setup, SOTA tools, Git workflow | Antigravity IDE, Cursor, Windsurf, Roo Code |
| **2** | Backend | Firebase, PostgreSQL, security | Firebase CLI, Supabase, npm audit |
| **3** | Automation | CI/CD, zero-downtime deployment | GitHub Actions, Docker, PM2 |
| **4** | Integration | Discord bots, Twilio, help desk | Discord.js, Twilio API, OpenClaw |

---

## Learning Objectives

By graduation, students will:

1. **Master SOTA 2026 CLI Tools** - Navigate 128+ modern Rust-based tools (television, succinctly, ugrep, qsv)
2. **Deploy Zero-Downtime Services** - Implement blue-green deployments with auto-rollback
3. **Build AI-Augmented Workflows** - Leverage Gemini CLI (Antigravity) for code generation and debugging
4. **Secure Production Infrastructure** - Automated vulnerability scanning, secret management, SSL automation
5. **Integrate Communication Systems** - Discord monitoring bots, Twilio alerts, help desk automation
6. **Automate Job Applications** - Resume parsing, AI cover letters, application tracking

## Instructor Notes

### Teaching Philosophy
- **Code First, Theory Second** - Students learn by building
- **SOTA Mandate** - Only teach current (2026) best practices
- **Security By Default** - Every module includes security considerations
- **Automation Focus** - If done twice, automate it

### Critical Success Factors
1. Complete Week 1 setup - no student left behind
2. Daily commits - enforce Git workflow
3. Production mindset - everything ships
4. Code review culture - learn from peers
